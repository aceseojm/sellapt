# 구글시트 컬럼 샘플 안내

## 파일 위치

- CSV 샘플: [구글시트_컬럼샘플_medispark.csv](/Users/jeongmin/Desktop/웹사이트/APT/구글시트_컬럼샘플_medispark.csv)

## 사용 방법

이 CSV 파일을 구글시트에 업로드하거나 첫 행부터 그대로 붙여넣으면 기본 컬럼 구조와 샘플 데이터가 함께 들어간다.

## 컬럼 설명

- `created_at`: 접수일시
- `name`: 고객 이름
- `phone`: 고객 연락처
- `developer`: 개발자명, 현재 고정값 `서정민`
- `page_name`: 랜딩페이지명
- `source`: 유입 소스 예: `direct`, `meta`, `naver`
- `campaign`: 광고 캠페인명
- `device`: 접속 디바이스 예: `mobile`, `desktop`
- `memo`: 상담 메모 또는 관심사항

## 첫 행 컬럼값

```text
created_at,name,phone,developer,page_name,source,campaign,device,memo
```

## 운영 메모

최소 수집만 필요하면 아래 4개 컬럼만 사용해도 된다.

- `created_at`
- `name`
- `phone`
- `developer`

광고 성과 추적까지 하려면 현재 샘플처럼 전체 컬럼 구조를 유지하는 것을 권장한다.
